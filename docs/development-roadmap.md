# Development Roadmap

1. Initialize monorepo with `frontend`, `backend`, `database`, `docs`, `docker`.
2. Build FastAPI domain models + SQLAlchemy persistence.
3. Implement risk formulas and underwriting APIs.
4. Add OCR + LLM extraction pipeline for uploaded documents.
5. Add LangChain multi-agent orchestration and decision explanation.
6. Store and retrieve underwriting context in local Chroma vector DB.
7. Build Next.js dashboard with intake form, charts, decisions, simulator.
8. Implement PDF report generation endpoint and frontend export.
9. Add Docker Compose for local full-stack runtime (including Ollama).
10. Add CI pipeline and cloud deployment configs for Vercel + AWS/GCP.
